# Procesamiento Paralelo con [[Procesos]]

> ¿Qué pasa si cada proceso tiene que hacer algo distinto?

Eviten en la medida de lo posible que los procesos usen scanf o printf. El proceso fue creado para procesar y morirse. Usar el printf o scanf es pérdida de tiempo, le quitan eficiencia al procesamiento.

Cuando se guardan los procesos del archivo, no necesariamente ser harán en orden. Todos los procesos interactúan independienemente entre ellos, a lo mejor el primer proceso que el sistema deje pasar no será el primero. No se vayan a espantar si eso paasa, eso es normal. Porque el procesamientva a depender del sistema operativo.
