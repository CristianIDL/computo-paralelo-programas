## Memoria Compartida 01

Códigp de "MemoriaCompartidaa01.c":

```
int main()
{
   int shmid1, shmid2, i;
   int *arreglo, *turno;
   key_t llave1, llave2;
   pid_t pid;
   llave1=ftok("Archivo1",'k');
   llave2=ftok("Archivo2",'k');
   
   shmid1=shmget(llave1,10*sizeof(int),IPC_CREAT|0777);
   shmid2=shmget(llave2,sizeof(int),IPC_CREAT|0777);
   
   arreglo=(int *)shmat(shmid1,0,0);
   turno=(int *)shmat(shmid2,0,0);

   *turno=0;
   for(i=0;i<10;i++)
   {
      arreglo[i]=i+1;
      printf("\nProceso escritor dice: %d", arreglo[i]);
      while(*turno!=1);
      *turno=0;
      sleep(1);
   }
   
   printf("\n\nEl escritor ha terminado\n\n");
   shmdt(&arreglo);
   shmdt(&turno);
   shmctl(shmid1,IPC_RMID,0);
   shmctl(shmid2,IPC_RMID,0);
   return 0;
}

```

Arreglo es para la memoria compartida con `shmat`, ahí se hace un _casteo_ (así le dicen los de sistemas pero el profe dice que le suena a castidad, entonces se le hace feo)


`arreglo=(int *)shmat(shmid1,0,0);`
`turno=(int *)shmat(shmid2,0,0);`

Cuando le ponen `0,0` es toda.

## Preguntas

Johnny pregunta de por qué el file to key

> Seguramente hay un algoritmo detrás que sea algún algoritmo como de cifrado o firma electrónica llamada `llave1`, entonces todos requerirán esas llaves para accedere al semáforo 1.

> `shmid` no me dice donde está

Le preguntaron sonbre esta línea: 
`shmid1=shmget(llave1,10*sizeof(int),IPC_CREAT|0777);`

> Lo que yo ponga en el arregelo se colocará en la [[RAM]] automáticamente de una forma muy rápida.

_En esencia sí es un poco como `malloc`, pero con diferente objetivo._

Lo que yo coloque en el arreglo en turno es algo que todos los procesis van a ver.

El primer proceso de todos es un abusivo. Porque él será el primero en forzar el `*turno` a `1`.

Al entrar en el while verifica cuanto vale el turno, si es diferente de 1 entonces lo dejará. 

> De hecho este algoritmo se usaba para semáforos antes de que [[E.W Djixtra]] implementara los semáforos

Aquí ya estamos en en terreno de memoria compartida. Al fin y al cabo un arreglo es un apuntador.
